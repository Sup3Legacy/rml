In this lesson, we will program a ball which bounces on the limits of a box and which is duplicated each time we click into the box.

We first define the data structure which represents the bounds of a box: 

type box =
    { left: float;
      right: float;
      top: float;
      bot: float; }

and we create a box: 

let box =
  { left = 0.; right = 400.;
    bot = 0.; top = 400.; }
