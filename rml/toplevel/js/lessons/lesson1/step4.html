Parallel composition

The notion of instant becomes really visible when parallel computations occurs. The parallel execution of two expressions is denoted e1 || e2 (the boolean disjuction is denoted e1 or e2). It guarantees that the two expressions are executed at each instant. 

#exec (   run (non_instantaneous_loop 10)
       || run (non_instantaneous_loop 10) );;

In the previous example, we can observe that each instance of the process non_instantaneous_loop prints one number per instant. In the following example, the function instantaneous_loop is executed in one instant (since it is not a process, there is no run to apply it). 

#exec (   run (non_instantaneous_loop 10)
       || instantaneous_loop 10 );;

We can observe now that when instantaneous_loop starts its execution it do not step before it terminates.
