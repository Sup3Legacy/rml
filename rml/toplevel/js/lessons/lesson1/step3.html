Communication

The communication between parallel processes is made through events that are broadcasted instantaneously. Hence, an event is present during the instant where it is broadcasted, otherwise it is absent.

An event is declare with the construct [signal].

signal s;;


The construct [await s] waits the next instant where s is broadcasted.

let process p = await s; print_endline "Hello!" ;;


#run p;;


An event is broadcasted using [emit].

emit s;;
