Logical time and parallel composition

ReactiveML is based on the synchronous model. In this model, time is a succession of instants. Any OCaml function is considered to be instantaneous.

let instantaneous_loop n = for i = 1 to n do print_int i; print_newline () done ;;


instantaneous_loop 10;;


Functions that can be executed through several instants are processes. The [pause] statement waits for the end of instant.

let process non_instantaneous_loop n = for i = 1 to n do print_int i; print_newline (); pause done ;;


#run (non_instantaneous_loop 10);;


The notion of instant is really illustrated when parallel computations occurs. The parallel execution of two expressions is denoted [e1 || e2]. It guaranties that the two expressions are executed at each instant.

let process ex1 = run (non_instantaneous_loop 10) || run (non_instantaneous_loop 10) ;;


#run ex1;;


In the previous example ([ex1]), we can observe that each instance of the process [non_instantaneous_loop] prints one number per instant. In the following example, the function [instantaneous_loop] is executed in one instant.

let process ex2 = run (non_instantaneous_loop 10) || instantaneous_loop 10 ;;


#run ex2;;


Remark: You can use the button "suspend" (or use the directive [#suspend;;]) to execute the program in a step by step mode. The to return into the sampled mode you can use the button "resume" (or the directive [#resume;;]).
