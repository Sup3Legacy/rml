OCaml code

ReactiveML is an extension of OCaml. Therefore, OCaml phrases are also ReactiveML phrases. For example:

1 + 2 ;;

[ "a" ; "b" ; "c" ] ;;

(fun x -> x + 1) 3 ;;

let a = 1 ;;

let sqr x = x *. x ;;

type 'a tree = | Leaf of 'a | Node of 'a * 'a tree * 'a tree ;;

let rec depth t = match t with | Leaf _ -> 1 | Node (_, t1, t2) -> 1 + max (depth t1) (depth t2) ;;

let n = depth (Node (1, Node (2, Leaf 4, Leaf 5), Leaf 3)) ;;

Functions of the OCaml standard library are also available.

List.rev [ "a" ; "b" ; "c" ] ;;
