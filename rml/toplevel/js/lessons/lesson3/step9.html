To use this process, we need a signal on which we can send and receive a process. In case of multiple emission during the same instant, we build the parallel composition of the sent processes. 

signal new_behavior
  default (fun state -> process ())
  gather (fun p q state ->
            process (run (p state) || run (q state)))
;;

Now we can execute a moving point whose behavior can be replaced dynamically. 

#run replace moving_point (new_state ()) new_behavior;;
