Now to replace this point moing left to right, we define behavior of a point going up and down. 

let process up_down state =
  for i = 1 to 50 do emit state.vel (0., 2.); pause done;
  loop
    for i = 1 to 100 do emit state.vel (0., - 2.); pause done;
    for i = 1 to 100 do emit state.vel (0., 2.); pause done
  end

A point going up and down is thus defined as follows. 

let process moving_point' state =
   run (up_down state) ||
   run (compute_pos state) || run (draw state)

To reconfigure the previous point, we only have to send this process on the signal new_behavior. 

emit new_behavior moving_point';;

We can restore the previous behavior. 

emit new_behavior moving_point;;
