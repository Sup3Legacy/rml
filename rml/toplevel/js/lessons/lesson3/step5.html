The behavior of a moving point is the parallel composition of the three preceding processes. 

let process moving_point state =
   run (compute_pos state)
   ||
   run (left_right state)
   ||
   run (draw state)


#run moving_point (new_state ());;

The created process can not be removed form the execution since it do not terminate. In the next step, we will see how to implement a simple process manager that can remove stop the execution of a process in a modular way.
