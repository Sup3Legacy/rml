Let see now how we can change that behavior of a process. We define a process replace which takes as argument an initial behavior p_init, a state state and a signal new_behavior on which a new behavior that must replace the current behavior can be sent. 

let rec process replace p_init state new_behavior =
  do
    run (p_init state)
  until new_behavior(p) ->
    run (replace p state new_behavior)
  done
