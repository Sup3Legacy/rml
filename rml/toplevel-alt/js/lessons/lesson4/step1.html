In this lesson, we use automata to implement a simple alarm-clock

The controller of an alarm-clock can be described by a three-states automaton.

  • Idle: no alarm is set;
  • Armed: waiting for the deadline;
  • Bipbipip: the clock is ringing.

Lets start with a very simple process that wait for a duration expressed in number of logical steps. 

let process sleep d = 
  for i = 1 to d do
    pause;
  done
